Accessing the Server, File and Client info using PHP 
and displaying it in a tabular form.

- pre-requisites
	- install xampp
	- save server-info.php under xampp->htdocs->a new folder of your choice-> 
	- start the xampp server (open xampp->Click start against the Apache module)
	- open the browser and type -> http://localhost/WT/server-info.php
		(here I've placed the file inside a folder called WT in htdocs)
